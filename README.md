# Projeto da disciplina "SCC0640-Base de Dados" da Universidade de São Paulo

Os códigos foram desenvolvidos em ambiente Linux (Debian). Para rodar a aplicação, utilizar os comandos do arquivo Makefile no diretório "Application":

- make check: checa se PostgreSQL está ativo no sistema.
- make all: compila os arquivos em C++.
- make run: roda o arquivo executável.

As credenciais para conexão com a base de dados devem ser designadas dentro do próprio Makefile.
